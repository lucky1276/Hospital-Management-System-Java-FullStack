# 🏥 Hospital Management System

A web-based Hospital Management System developed using Java, JSP, Servlets, JDBC, and MySQL.

## 🚀 Features

- User Login & Logout
- Patient Management (CRUD)
- Doctor Management (CRUD)
- Appointment Management (CRUD)
- Billing Management (CRUD)
- Dashboard with Statistics
- Search Patients
- Search Doctors
- Search Bills
- Session Management

## 🛠 Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat 10
- Eclipse Enterprise Edition
