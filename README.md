# 🏥 Hospital Management System

A Hospital Management System developed using Java Full Stack technologies. This application allows users to register, log in, and manage patient records through CRUD operations.

## 🚀 Features

- User Registration
- User Login
- Add Patient
- View Patients
- Update Patient
- Delete Patient
- MySQL Database Integration
- Session Management

## 🛠 Technologies Used

- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat
- Eclipse IDE

## 📂 Project Structure

```
src/
 ├── main/
 │   ├── java/
 │   ├── webapp/
```

## 💻 How to Run

1. Clone the repository.
2. Import it into Eclipse.
3. Configure Apache Tomcat.
4. Create the MySQL database.
5. Run the project.
6. Open it in your browser.
